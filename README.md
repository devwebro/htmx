## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser and navigate to:
```
http://localhost:5000
```

## Technologies

- **Backend**: Python Flask
- **Frontend**: HTML, htmx, Bootstrap 5
- **Styling**: Custom CSS with gradient backgrounds

## How it works

- htmx handles AJAX requests without writing JavaScript
- Flask serves the backend API and templates
- Bootstrap provides responsive styling
- Todos are stored in memory (resets on server restart)
